## 🎓 Về Chứng Chỉ PCEP™ (Certified Entry-Level Python Programmer)

Chứng chỉ **PCEP™** đánh giá và chứng minh khả năng của ứng viên trong việc hoàn thành các nhiệm vụ lập trình cơ bản bằng ngôn ngữ Python. 

Người sở hữu chứng chỉ này được công nhận là đã hiểu rõ và có khả năng vận dụng tốt:
* Các khái niệm phổ quát của lập trình máy tính.
* Cú pháp và ngữ nghĩa của ngôn ngữ Python.
* Môi trường thực thi (runtime environment).
* Kỹ năng giải quyết các thách thức lập trình điển hình với sự trợ giúp của Thư viện Chuẩn (Python Standard Library).

## 📚 Phạm Vi Kiến Thức Được Bao Phủ

Kho mã nguồn này chứa các ví dụ và bài thực hành xoay quanh các chủ đề trọng tâm của kỳ thi PCEP:

- [x] **Các khái niệm cơ bản:** Thuật ngữ lập trình (biên dịch vs. thông dịch), logic và cấu trúc chương trình Python (từ khóa, lệnh, thụt lề).
- [x] **Kiểu dữ liệu & Biến:** Khai báo biến, hằng số, hệ thống số, và các kiểu dữ liệu cơ bản.
- [x] **Toán tử & Biểu thức:** Phép toán số học, logic (Boolean), so sánh, v.v.
- [x] **Thao tác I/O:** Nhập/xuất dữ liệu cơ bản với màn hình console.
- [x] **Điều khiển luồng (Control Flow):** Các khối lệnh điều kiện (`if`, `elif`, `else`) và vòng lặp (`for`, `while`, `break`, `continue`).
- [x] **Cấu trúc dữ liệu (Data Collections):** Thao tác với List, Tuple, Dictionary và String.
- [x] **Hàm (Functions):** Hàm tích hợp (built-in), hàm do người dùng định nghĩa, phân rã bài toán, tổ chức tương tác môi trường, trình tạo (generators) và đệ quy.
- [x] **Xử lý Ngoại lệ (Exceptions):** Cấu trúc phân cấp ngoại lệ và cách bắt lỗi (`try`, `except`).

## 📂 Cấu Trúc Kho Lưu Trữ

Dự án được phân chia thành các tệp lệnh và thư mục thực hành riêng biệt để dễ dàng tra cứu:

* `LAB_*.py`: Các bài tập thực hành theo từng chủ đề cụ thể (VD: `LAB_break.py`, `LAB_breakcontinue.py`, `LAB_Magican.py`, `LAB_binaryLR.py`).
* `Boolean_comparison.py`, `GettingHelp.py`: Các tệp mã nguồn minh họa cú pháp cơ bản, cách dùng biểu thức logic và các hàm hỗ trợ.
* `MinMax/`: Thuật toán tìm kiếm giá trị lớn nhất, nhỏ nhất.
* `Code&DataSet/`: Chứa các đoạn mã lưu trữ và dữ liệu bài tập đi kèm.
* Các thư mục mở rộng (`dtree_example/`, `nb_example/`, `SWI-Prolog/`): Dữ liệu và ví dụ ứng dụng thuật toán nâng cao được tích hợp thêm trong quá trình học.

## 🚀 Hướng Dẫn Sử Dụng

1. **Tải mã nguồn về máy:**
   ```bash
   git clone [https://github.com/tamton23/Python-Basic-certificate.git](https://github.com/tamton23/Python-Basic-certificate.git)
   cd Python-Basic-certificate
   ```

2. **Chạy thử một bài Lab:**
   Bạn có thể sử dụng bất kỳ IDE nào (như VS Code, PyCharm) hoặc chạy trực tiếp thông qua terminal:
   ```bash
   python LAB_breakcontinue.py
   ```
